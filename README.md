# Student Onboarding Landing Page

React + Vite frontend for the first-login student onboarding flow.

## Requirements

- Node.js 18+
- npm

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

Production files are generated in `dist/`.
